# Welcome to mySQL-Employee-Tracker 👋

> A command line application that allows users to: add departments,roles,employees// view departments, roles, employees// update employee roles


## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```
npm install
```

## Usage

```
node app.js
```

## Run tests

```
npm run test
```
